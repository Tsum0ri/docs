## Overview

ServerManagerPro provides a comprehensive set of commands for server administration, moderation, and player management. All commands support cross-server execution in a networked environment.

\<Note\>

  All permissions support wildcards. For example, `servermanager.admin.*` grants all admin permissions.

  The special permissions `OP` and `*` grant full access to all commands.

\</Note\>

## Dynamic Server Commands

Server transfer commands are dynamically registered based on your `network.json` configuration.

\<Info\>

  Each server defined in your network config with a `command` field will have a transfer command registered.

  For example, if you have a server with `"command": "hub"`, players can use `/hub` to transfer.

\</Info\>

**Permission format:** `servermanager.server.<command>`

**Example:** `/hub` requires `servermanager.server.hub`
